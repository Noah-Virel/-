🔧 GAS Web App URL を設定

Google Apps Script でデプロイした Web App の URL を入力してください。
設定は次回以降も保持されます。

📋 GAS デプロイ手順：
① スプレッドシートのメニュー → 「拡張機能」→「Apps Script」
② 提供された gas_code.js の中身を貼り付けて保存
③ 右上 「デプロイ」→「新しいデプロイ」
④ 種類：ウェブアプリ ／ アクセス：全員 を選択
⑤ 発行された URL（https://script.google.com/macros/s/...）を下に貼り付け
